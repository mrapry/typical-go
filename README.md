# Typical Go

A Framework and Build Tool for Productive Go Development. <https://typical-go.github.io/>

## Install

*On Progress*

## Usage

### New Project

```bash
typical-go new [PACKAGE]
```

### Create Wrapper

```bash
typical-go wrapper
typical-go wrapper -path=[Path]
```
